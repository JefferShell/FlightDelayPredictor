## 航班离港延误系统

### 项目简介

航班离港延误系统是一个基于预测算法的系统，主要用于展示航班延误的相关信息。系统自带航班数据，可以直接初始化进入，目前是sqlite可以改成mysql，根据已有航班数据，使用神经网络模型训练了数据，最后产出一个模型保存下来，后面每次根据具体的航班信息，调用模型可以做延误预测。

### 主要功能

- 显示航班延误的相关信息
- 提供航班数据查询功能
- 利用神经网络模型预测航班延误
- 提供用户与系统管理员之间的交流平台


### 技术架构

- 前端：HTML、CSS、JavaScript
- 后端：Python、Flask
- 数据库：SQLite


### 使用说明

1. 下载并解压项目包
2. 使用虚拟环境安装依赖项
3. 运行项目并访问`localhost:5000`
